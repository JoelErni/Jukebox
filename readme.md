# Python Jukebox with MongoDB

## Software and Technologies
- Visual Studio Code
- **Python3.11**
- MongoDB

## Modules
- `re`
- `json`
- `bson`
- `playsound`
- `pymongo`

## Instructions
### Upload Song
1. Put the `.mp3`-file in the `export_songs` folder
2. Run the `player.py`
3. Select `Add song`(0)
4. Enter the songname, Filename etc..

### Read Song
1. Run `player.py`
2. Select `Search song`
3. Enter a keyword (optional)
4. Select a song
5. Select an operation
